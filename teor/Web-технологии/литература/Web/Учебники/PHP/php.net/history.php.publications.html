Публикации о PHP

Насколько нам известно, первая статья о PHP появилась в журнале French Informatiques Magazine в конце 1998 года и была посвящена PHP 3.0. Это была первая статья в серии статей, опубликованных о PHP в различных журналах.

Статьи о PHP появлялись в Dr. Dobbs, Linux Enterprise, Linux Magazine и во многих других изданиях. Статьи о переводе приложений на основе ASP в PHP под Windows появляются даже в собственной библиотеке MSDN фирмы Microsoft!

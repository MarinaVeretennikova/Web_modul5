Как мы генерируем форматы

Этот учебник написан на XML с использованием DocBook XML DTD, DSSSL (Document Style and Semantics Specification Language) для форматирования и, в качестве эксперимента, XSLT (Extensible Stylesheet Language Transformations) - для обработки и трансформации.

Использование XML в качестве исходного формата даёт нам возможность генерировать различные выводные форматы, обслуживая только один исходник для всех форматов.
Для форматирования HTML и TeX версий используются утилиты: Jade, написанная James'ом Clark'ом, и The Modular DocBook Stylesheets Norman'a Walsh'a.
Мы использовали Microsoft HTML Help Workshop для генерации формата Windows HTML Help учебника и, конечно, сам PHP - для выполнения различных дополнительных конвертаций и форматирования.

Вы можете загрузить этот учебник в различных форматах и на разных языках, включая обычный текст, HTML, PDF, PalmPilot DOC, PalmPilot iSilo и Windows HTML Help, с сайта http://www.php.net/docs.php.
Эти учебники обновляются автоматически по мере обновления исходного материала.

Вы можете найти дополнительную информацию о загрузке исходного XML-кода этой документации по адресу http://cvs.php.net/. Документация хранится в модуле phpdoc.

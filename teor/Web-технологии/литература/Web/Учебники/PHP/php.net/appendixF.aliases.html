Приложение F. Список псевдонимов функций

Это список всех псевдонимов. Обычно не стоит использовать псевдонимы, так как они могут быть связаны с устаревшими или переименованными функциями, что даст непереносимый код. Это список дан для того, чтобы помочь обновить старые скрипты новым синтаксисом.

Однако некоторые функции имеют два имени, и невозможно отдать предпочтение одному из них. (Например, is_int() и is_integer() равноценны).

Этот список соответствует PHP 4.0.6.

Где найти информацию о PHP

Этот учебник не пытается дать инструкции по общим вопросам программирования. Если вы - начинающий программист, для вас может оказаться затруднительным изучения программирования на PHP только с помощью данного учебника. Возможно, вам понадобится информация для начинающих. Список книг, имеющих отношение к PHP, можно найти по адресу: http://www.php.net/books.php.

Имеются несколько действующих списков рассылки/mailing lists, где обсуждаются все аспекты программирования на языке PHP. Если вы не можете найти сами решение какой-либо проблемы, вы можете получить помощь со стороны кого-нибудь из участников этих списков. Списки рассылки находятся по адресу: http://www.php.net/support.php, где имеются также ссылки на архивы переписки и другие ресурсы online-поддержки. Помимо этого, на http://www.php.net/links.php имеется список web-сайтов, где размещены статьи, форумы и галереи кода PHP.

Форматы

Учебник PHP имеется в различных форматах. Эти форматы можно разделить на две группы: online-форматы и загружаемые пакеты.

Примечание: Некоторые издательства сделали печатные версии этого учебника. Мы не может рекомендовать ни одну из них, так как они очень быстро устаревают.

Учебник можно читать online по адресу http://www.php.net/ и на многочисленных сайтах-зеркалах. Для большей скорости выберите ближайший сайт. Вы может читать учебник в обычном (удобном для печати) HTML-формате или в HTML-формате, интегрирующем этот учебник в единое целое.

Преимущество online-учебника в том, что в нём встроены замечания пользователей. Очевидным недостатком является необходимость подключения к сети.

Есть несколько несетевых форматов этого учебника, и можно выбрать наиболее подходящий для вас и вашей ОС. О том, как учебник генерируется в такое количество форматов, см. раздел 'Как мы генерируем форматы' в этом приложении.

Самые платформонезависимые форматы это HTML и текстовая версия. Формат HTML представлен в одиночном HTML-файле и как пакет небольших отдельных файлов по каждому разделу (образуя коллекцию из более чем 3000 файлов). Форматы HTML и plaintext предоставлены как tar-файлы, сжатые с использованием архиватора bzip2.

Другой популярный межплатформенный формат, и более удобный для печати, это PDF (известный также как Adobe Acrobat). Но предупреждаем, (прежде чем вы загрузите этот формат и нажмёте кнопку Print), что в учебнике более 3000 страниц, и он постоянно пересматривается.

Примечание: если у вас пока нет программы для просмотра файлов в формате PDF, вам нужно загрузить программу Adobe Acrobat Reader.

Для владельцев портативных Palm-совместимых устройств подходят форматы документов Palm и iSilo. Вы можете носить портативное устройство с собой постоянно и использовать программу DOC или iSilo для изучения PHP, или просто использовать устройство как портативный справочник.

Для платформ Windows - версия Windows HTML Help (.СНМ-файл) этого учебника соединяет файлы формата HTML для использования в приложении Windows HTML Help. В этой версии имеются полнотекстовый поиск, полнотекстовый индекс и закладки. Многие популярные среды разработки РНР под Windows также интегрируются с этой версией документации для облегчения доступа к информации.

Примечание: проект Visual Basic для Linux находится в стадии планирования и будет включать разработку CHM Creator и Viewer для Linux. См. страницу SourceForge.net, если вас интересует этот процесс.

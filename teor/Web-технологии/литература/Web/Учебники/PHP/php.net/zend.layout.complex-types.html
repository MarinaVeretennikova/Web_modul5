Сложные (комплексные) типы

Сложные типы, такие как массивы и объекты, требуют иного подхода. Zend предлагает единый API для этих типов - они хранятся с использованием хэш-таблиц.

Примечание: чтобы упростить последующие примеры, мы работаем только с простыми типами, в первую очередь с такими, как integer. Обсуждение создания более продвинутых типов идёт далее в этой же главе.

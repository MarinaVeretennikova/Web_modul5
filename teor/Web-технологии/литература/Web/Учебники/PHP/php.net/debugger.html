Об отладчике

В PHP 3 имеется поддержка сетевого отладчика.

В PHP 4 такой возможности нет. Но вы можете использовать один из внешних отладчиков. В Zend IDE имеется отладчик, имеются также несколько свободно распространяемых отладочных расширений наподобие DBG по адресу http://dd.cron.ru/dbg/ или Advanced PHP Debugger (APD).

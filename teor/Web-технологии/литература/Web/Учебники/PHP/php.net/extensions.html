Расширения PHP 3

Расширения, написанные для PHP 3, не будут работать в PHP 4 ни как двоичники, ни как исходники. Нетрудно перенести расширения в PHP 4, если у вас есть доступ к оригинальному коду.
Детальное описание процесса переноса не является частью данного текста.

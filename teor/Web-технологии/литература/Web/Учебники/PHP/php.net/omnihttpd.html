Серверы OmniHTTPd

В этом разделе дана информация по OmniHTTPd.

OmniHTTPd 2.0b1 и новее под Windows

Вам необходимо выполнить следующие шаги, чтобы PHP заработал с OmniHTTPd. Это установка CGI-исполняемого файла. SAPI поддерживается в OmniHTTPd, но некоторые тесты показали, что использование PHP как ISAPI-модуля нестабильно.

  • Шаг 1: Установите OmniHTTPd-сервер.

  • Шаг 2: Щёлкните правой кнопкой мыши на голубой иконке OmniHTTPd в системном отделе и выберите Properties

  • Шаг 3: Щёлкните Web Server Global Settings

  • Шаг 4: На вкладке 'External' введите: virtual = .php | actual = c:\path-to-php-dir\php.exe и используйте кнопку Add.

  • Шаг 5: На вкладке Mime введите: virtual = wwwserver/stdcgi | actual = .php и используйте кнопку Add.

  • Шаг 6: Нажмите OK.

Повторите шаги 2 - 6 для каждого расширения, которое вы хотите ассоциировать с PHP.

Примечание: некоторые OmniHTTPd-пакеты поставляются со встроенной поддержкой PHP. В ходе выполнения установки вы может избрать специальную/custom установку и снять выделение PHP-компонента. Мы рекомендуем использовать самые последние исполняемые PHP. Некоторые OmniHTTPd-серверы поставляются с PHP 4 beta-дистрибутивами, так что вам не нужно устанавливать встроенную поддержку, а установить свою. Если сервер уже находится на вашей машине, используйте кнопку Replace в Шагах 4 и 5 для установки новой корректной информации.

Объекты

Инициализация объекта

Для инициализации объекта используется оператор new для инстанциации объекта в переменной.

<?php
class foo
{
    function do_foo()
    {
        echo "Doing foo."; 
    }
}

$bar = new foo;
$bar->do_foo();
?>

Полное рассмотрение производится в разделе Классы и Объекты.

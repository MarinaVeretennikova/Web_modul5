Функции работы с файлами и директориями

Следующие функции для работы с файлами и директориями должны использоваться в Zend-модулях (они ведут себя точно так же, как их C-двойники):
Zend-функцияРегулярная C-функция
V_GETCWD()getcwd()
V_FOPEN()fopen()
V_OPEN()open()
V_CHDIR()chdir()
V_GETWD()getwd()
V_CHDIR_FILE()Принимает путь к файлу как аргумент и делает текущей рабочей директорией директорию этого файла.
V_STAT() stat()
V_LSTAT() lstat()

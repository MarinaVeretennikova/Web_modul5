Предопределённые константы oracle - константы, определённые в расширении oracle.

Описание

Эти константы определены в расширении oracle и будут доступны только в том случае, если это расширение будет скомпилировано с PHP или динамически загружено на этапе прогона.

ORA_BIND_INOUT (integer)
ORA_BIND_IN (integer)
ORA_BIND_OUT (integer)
ORA_FETCHINTO_ASSOC (integer)
ORA_FETCHINTO_NULLS (integer)

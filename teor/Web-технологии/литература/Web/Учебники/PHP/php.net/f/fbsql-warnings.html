fbsql_warnings

(PHP 4 >= 4.0.6)

fbsql_warnings - включает или отключает предупреждения FrontBase.

Описание

bool fbsql_warnings ([bool OnOff])

Возвращает TRUE, если предупреждения включены, иначе FALSE.

fbsql_warnings() включает или отключает предупреждения FrontBase.

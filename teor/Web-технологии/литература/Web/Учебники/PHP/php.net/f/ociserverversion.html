OCIServerVersion

(PHP 3>= 3.0.4, PHP 4)

OCIServerVersion - возвращает строку с информацией о версии сервера.

Описание

string OCIServerVersion (int conn)

Пример 1. OCIServerVersion
<?php
   $conn = OCILogon("scott","tiger");
   print "Server Version: " . OCIServerVersion($conn);
   OCILogOff($conn);
?>

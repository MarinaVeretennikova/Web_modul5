ovrimos_result

(PHP 4 >= 4.0.3)

ovrimos_result - запрашивает столбец вывода.

Описание

int ovrimos_result (int result_id, mixed field)

ovrimos_result() запрашивает столбец вывода 'field' как строку или как индекс с базой 1.

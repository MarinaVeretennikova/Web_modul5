headers_sent

(PHP 3>= 3.0.8, PHP 4)

headers_sent - возвращает TRUE, если шапки/headers отправлены.

Описание

bool headers_sent (void)

Эта функция возвращает TRUE, если HTTP headers уже отправлены, FALSE в ином случае.

См. также header().

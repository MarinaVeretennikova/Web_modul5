crc32

(PHP 4 >= 4.0.1)

crc32 - вычисляет crc32-полином строки.

Описание

int crc32 (string str)

Генерирует циклически избыточный checksum-полином 32-битных длин str. Это обычно используется для проверки целостности передаваемых данных.

См. также md5().

array_fill

(PHP 4 >= 4.2.0)

array_fill - заполняет массив значениями.

Описание

array array_fill (int start_index, int num, mixed value)

array_fill() заполняет массив num количеством вхождений значения параметра value, ключами, начиная с параметра start_index.

Пример 1. array_fill()
$a = array_fill(5, 6, 'banana');

$a теперь имеет следующие вхождения, с помощью print_r():
Array
(
    [5]  => banana
    [6]  => banana
    [7]  => banana
    [8]  => banana
    [9]  => banana
    [10] => banana
)

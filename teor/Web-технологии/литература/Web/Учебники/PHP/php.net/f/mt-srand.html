mt_srand

(PHP 3>= 3.0.6, PHP 4)

mt_srand - сеет наилучший генератор случайных чисел.

Описание

void mt_srand (int seed)

Производит посев генератора случайных чисел с seed. 
// сеет с микросекундами
function make_seed() {
    list($usec, $sec) = explode(' ', microtime());
    return (float) $sec + ((float) $usec * 100000);
}
mt_srand(make_seed());
$randval = mt_rand();

Примечание: начиная с PHP 4.2.0, больше нет необходимости производить посев генератора случайных чисел перед его использованием.

См. также mt_rand(), mt_getrandmax() и srand().

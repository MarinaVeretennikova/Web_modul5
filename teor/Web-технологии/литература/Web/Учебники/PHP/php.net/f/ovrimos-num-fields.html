ovrimos_num_fields

(PHP 4 >= 4.0.3)

ovrimos_num_fields - возвращает количество столбцов.

Описание

int ovrimos_num_fields (int result_id)

ovrimos_num_fields() возвращает количество столбцов в result_id, результирующем из запроса.

closedir

(PHP 3, PHP 4)

closedir - закрывает дескриптор директории.

Описание

void closedir (resource dir_handle)

Закрывает поток директории, обозначенный параметром dir_handle. Этот поток обязан быть ранее открыт функцией opendir().

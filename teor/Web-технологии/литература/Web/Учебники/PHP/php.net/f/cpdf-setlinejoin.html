cpdf_setlinejoin

(PHP 3>= 3.0.8, PHP 4)

cpdf_setlinejoin - устанавливает параметр linejoin.

Описание

void cpdf_setlinejoin (int pdf document, long value)

Функция cpdf_setlinejoin() устанавливает параметр linejoin в значение от 0 до 2.

0 = miter, 1 = round, 2 = bevel.

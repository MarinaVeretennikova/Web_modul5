imagecreatefromxbm

(PHP 4 >= 4.0.1)

imagecreatefromxbm - создаёт новое изображение из файла или URL.

Описание

resource imagecreatefromxbm (string filename)

imagecreatefromxbm() возвращает идентификатор изображения полученного из данного файла filename.

apache_setenv

(PHP 4 >= 4.2.0)

apache_setenv - устанавливает Apache-переменную subprocess_env.

Описание

int apache_setenv (string variable, string value [, bool walk_to_top])

Предупреждение!

Эта функция в настоящее время не задокументирована; имеется только список аргументов.

ob_get_level

(PHP 4 >= 4.2.0)

ob_get_level - возвращает содержащий уровень механизма буферизации вывода.

Описание

int ob_get_level (void)

Возвращает уровень вложенных обработчиков вывода.

См. также ob_start() и ob_get_contents().

sqrt

(PHP 3, PHP 4)

sqrt - квадратный корень.

Описание

float sqrt (float arg)

Возвращает квадратный корень аргумента arg.

<?php
// Точность зависит от вашей precision-директивы
echo sqrt(9); // 3
echo sqrt(10); // 3.16227766 ...
?>

См. также pow().

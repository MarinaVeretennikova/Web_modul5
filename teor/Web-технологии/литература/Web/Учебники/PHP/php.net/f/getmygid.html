getmygid

(PHP 4 >= 4.1.0)

getmygid - получает GID владельца РНР-скрипта.

Описание

int getmygid (void)

Возвращает group ID текущего скрипта или FALSE при ошибке.

См. также getmyuid(), getmypid(), get_current_user(), getmyinode() и getlastmod().

mcal_is_leap_year

(PHP 3>= 3.0.13, PHP 4)

mcal_is_leap_year - возвращает, является ли год високосным.

Описание

int mcal_is_leap_year (int year)

mcal_is_leap_year() возвращает 1, если данный год является високосным, 0 - если нет.

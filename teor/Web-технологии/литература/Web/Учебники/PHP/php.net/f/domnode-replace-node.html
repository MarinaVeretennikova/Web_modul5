DomNode->replace_node - заменяет узел.

Описание

object DomNode->replace_node (object newnode)

Эта функция заменяет существующий узел новым переданным. Перед замещением newnode копируется, если имеет предка, чтобы гарантировать, что узел, который который уже имеется в документе, не будет вставлен вторично. Это поведение форсирует все модификации узла до замещения или повторное получение вставляемого узла затем с помощью функций вроде DomNode_first_child(), DomNode_child_nodes() etc.

См. также DomNode_append_child().

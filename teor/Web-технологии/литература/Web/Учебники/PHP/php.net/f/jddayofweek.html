JDDayOfWeek

(PHP 3, PHP 4)

JDDayOfWeek - возвращает день недели.

Описание

mixed jddayofweek (int julianday, int mode)

Возвращает день недели. Может возвратить строку или integer, в зависимости от режима.

Таблица 1. Режимы календарной недели
РежимЗначение
0Возвращает номер дня как int (0=sunday, 1=monday, и т.д.)
1Возвращает строку, содержащую день недели (english-gregorian)
2 Возвращает строку, содержащую аббревиатуру дня недели (english-gregorian)

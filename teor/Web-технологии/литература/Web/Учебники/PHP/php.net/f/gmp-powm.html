gmp_powm

(PHP 4 >= 4.0.4)

gmp_powm - возводит число в степень с modulo.

Описание

resource gmp_powm (resource base, resource exp, resource mod)

Вычисляет base, возведённую в степень exp, modulo mod. Если exp отрицательная, результат не определён.

java_last_exception_get

(PHP 4 >= 4.0.2)

java_last_exception_get - получает последнее Java-исключение.

Описание

exception java_last_exception_get (void)

Следующий пример демонстрирует использование обработчика Java-исключения в PHP:
Пример 1. Обработчик исключения Java
<?php
  $stack = new Java('java.util.Stack');
  $stack->push(1);

  // Этот должно пройти
  $result = $stack->pop();
  $ex = java_last_exception_get();
  if (!$ex) print "$result\n";

  // Это не пройдёт (ошибка подавляется префиксом @)
  $result = @$stack->pop();
  $ex = java_last_exception_get();
  if ($ex) print $ex->toString();

  // Очистить последнее исключение
  java_last_exception_clear();
?>

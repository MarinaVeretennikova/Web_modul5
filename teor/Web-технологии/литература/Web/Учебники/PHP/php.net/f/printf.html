printf

(PHP 3, PHP 4)

printf - выводит форматированную строку.

Описание

void printf (string format [, mixed args])

Создаёт вывод в соответствии с format, который описан в документации к sprintf().

См. также print(), sprintf(), sscanf(), fscanf() и flush().

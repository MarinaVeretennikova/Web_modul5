imap_headers

(PHP 3, PHP 4)

imap_headers - возвращает шапки всех сообщений в mailbox'е.

Описание

array imap_headers (int imap_stream)

Возвращает массив отформатированной строки с header-информацией. Один элемента на mail-сообщение.

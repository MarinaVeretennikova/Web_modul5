srand

(PHP 3, PHP 4)

srand - сеет генератор случайных чисел.

Описание

void srand (int seed)

Сеет генератор случайных чисел с числом seed. 
// сеет с микросекундами
function make_seed() {
    list($usec, $sec) = explode(' ', microtime());
    return (float) $sec + ((float) $usec * 100000);
}
srand(make_seed());
$randval = rand();

Примечание: начиная с PHP 4.2.0, больше не нужно сеять генератор случайных чисел, перед тем как его использовать.

См. также rand(), getrandmax() и mt_srand().

pdf_set_font

(PHP 3>= 3.0.6, PHP 4)

pdf_set_font - не рекомендуется: выбирает шрифт и его размер.

Описание

Не рекомендуется. Вместо неё вы должны использовать pdf_findfont() плюс pdf_setfont().

См. pdf_findfont(), pdf_setfont().

phpcredits

(PHP 4)

phpcredits - печатает кредиты для PHP.

Описание

void phpcredits ([int flag])

Эта функция печатает список кредитов разработчиков, модулей etc. Она генерирует соответствующие HTML-коды для вставки информации на страницу. flag является необязательным и по умолчанию имеет значение CREDITS_ALL. Для генерации специализированной страницы кредитов вы можете использовать параметр flag. Например, для печати основных кредитов вы используете в вашем коде что-нибудь такое:
...
phpcredits(CREDITS_GENERAL);
...

Если вы хотите напечатать разработчиков ядра и группу документации, вы используете:

<?php
phpcredits(CREDITS_GROUP + CREDITS_DOCS + CREDITS_FULLPAGE);
?>

А если вам нужно внедрить на страницу все кредиты, это сделает примерно такой код:

<html>
 <head>
  <title>My credits page</title>
 </head>
 <body>
  <?php
 // какой-нибудь ваш код
 phpcredits(CREDITS_ALL);
 // ещё код
 ?>
 </body>
</html>

Таблица 1. Предопределённые флаги phpcredits()
имяописание
CREDITS_ALL Все кредиты; эквивалентно использованию: CREDITS_DOCS + CREDITS_GENERAL + CREDITS_GROUP + CREDITS_MODULES + CREDITS_FULLPAGE. Генерирует полную отдельную HTML-страницу с соответствующими тэгами.
CREDITS_DOCSКредиты команды документации.
CREDITS_FULLPAGE Обычно используется в комбинации с другими флагами. Указывает, что необходимо напечатать полную отдельную HTML-страницу, включая информацию, указанную другими флагами.
CREDITS_GENERAL Главные кредиты: дизайн и концепция языка, авторы PHP 4.0 и SAPI-модуль.
CREDITS_GROUPСписок разработчиков ядра.
CREDITS_MODULES Список модулей расширений PHP и их авторов.
CREDITS_SAPI Список модулей серверного API PHP и их авторов.

См. также phpinfo(), phpversion() и php_logo_guid().

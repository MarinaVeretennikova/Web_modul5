gmp_intval

(PHP 4 >= 4.0.4)

gmp_intval - конвертирует GMP-число в целое.

Описание

int gmp_intval (resource gmpnumber)

Эта функция позволяет конвертировать GMP-число в целое.
Предупреждение!

Эта функция возвращает пригодный для использования результат только в том случае, если число действительно соответствует целому числу PHP (т.е. тип long со знаком). Если вы хотите просто напечатать GMP-число, используйте gmp_strval().

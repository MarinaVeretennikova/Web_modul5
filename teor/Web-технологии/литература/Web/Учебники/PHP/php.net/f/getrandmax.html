getrandmax

(PHP 3, PHP 4)

getrandmax - показывает наибольшее возможное случайное значение.

Описание

int getrandmax (void)

Возвращает максимальное значение, которое может быть возвращено вызовом функции rand().

См. также rand(), srand() и mt_getrandmax().

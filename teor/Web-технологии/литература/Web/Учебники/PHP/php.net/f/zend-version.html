zend_version

(PHP 4)

zend_version - получает текущую версию Zend-машины.

Описание

string zend_version (void)

Возвращает строку с версией текущего работающего PHP-разборщика.
Пример 1. zend_version()
// печатает e.g. 'Zend engine version: 1.0.4'
echo "Zend engine version: " . zend_version();

См. также phpinfo(), phpcredits(), php_logo_guid() и phpversion().

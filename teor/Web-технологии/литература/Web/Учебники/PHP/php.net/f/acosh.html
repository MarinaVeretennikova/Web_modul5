acosh

(PHP 4 >= 4.1.0)

acosh - инверсный гиперболический косинус.

Описание

float acosh (float arg)

Возвращает инверсный гиперболический косинус аргумента arg, т.е. значение, чей гиперболический косинус является arg.

Примечание: эта функция не реализована на платформах Windows.

См. также acos(), asinh() и atanh().

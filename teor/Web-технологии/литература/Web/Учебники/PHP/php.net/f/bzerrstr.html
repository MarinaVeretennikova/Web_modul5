bzerrstr

(PHP 4 >= 4.0.4)

bzerrstr - возвращает строку с ошибкой bzip2.

Описание

string bzerrstr (resource bz)

Возвращает строку ошибки любой bzip2-ошибки, возвращённой файловым указателем bz.

См. также bzerrno() и bzerror().

unixtojd

(PHP 4)

unixtojd - конвертирует UNIX timestamp в Julian Day.

Описание

int unixtojd ([int timestamp])

Возвращает Julian Day для UNIX timestamp (секунд после 1.1.1970) или для текущего дня, если timestamp не задан.

См. также jdtounix().

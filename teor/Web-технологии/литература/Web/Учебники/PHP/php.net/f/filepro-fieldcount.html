filepro_fieldcount

(PHP 3, PHP 4)

filepro_fieldcount - определяет, сколько полей имеется в БД filePro.

Описание

int filepro_fieldcount (void)

Возвращает количество полей (столбцов) в открытой БД filePro.

См. также filepro().

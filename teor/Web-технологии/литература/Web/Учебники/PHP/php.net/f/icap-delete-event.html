icap_delete_event

(PHP 4)

icap_delete_event - удаляет событие из ICAP-календаря.

Описание

string icap_delete_event (int stream_id, int uid)

icap_delete_event() удаляет календарное событие, специфицированное параметром uid.

Возвращает TRUE.

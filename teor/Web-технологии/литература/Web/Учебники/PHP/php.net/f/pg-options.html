pg_options

(PHP 3, PHP 4)

pg_options - получает опции, ассоциированные с соединением.

Описание

string pg_options (resource connection)

pg_options() возвращает строку, содержащую опции, специфицированные в данном PostgreSQL connection-ресурсе.

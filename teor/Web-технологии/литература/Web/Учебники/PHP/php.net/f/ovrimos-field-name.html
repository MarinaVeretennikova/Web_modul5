ovrimos_field_name

(PHP 4 >= 4.0.3)

ovrimos_field_name - возвращает имя столбца вывода.

Описание

int ovrimos_field_name (int result_id, int field_number)

ovrimos_field_name() возвращает имя столбца вывода по специфицированному индексу (с базой 1).

JDMonthName

(PHP 3, PHP 4)

JDMonthName - возвращает название месяца.

Описание

string jdmonthname (int julianday, int mode)

Возвращает строку, содержащую название месяца.
mode говорит этой функции, в какой календарь конвертировать Julian Day Count/Юлианское летосчисление и каков тип возвращаемых названий месяцев.

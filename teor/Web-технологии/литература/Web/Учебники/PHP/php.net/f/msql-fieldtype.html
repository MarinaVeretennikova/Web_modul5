msql_fieldtype

(PHP 3, PHP 4)

msql_fieldtype - получает тип поля.

Описание

string msql_fieldtype (int query_identifier, int i)

msql_fieldtype() аналогична msql_fieldname(). Аргументы идентичны, но возвращается тип поля. Это может быть "int", "char" или "real".

ftp_mkdir

(PHP 3>= 3.0.13, PHP 4)

ftp_mkdir - создаёт директорию/каталог.

Описание

string ftp_mkdir (resource ftp_stream, string directory)

Создаёт специфицированную directory.

Возвращает имя вновь созданной директории при успехе, FALSE - при ошибке.

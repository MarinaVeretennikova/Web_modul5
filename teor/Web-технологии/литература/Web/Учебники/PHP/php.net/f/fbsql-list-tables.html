fbsql_list_tables

(PHP 4 >= 4.0.6)

fbsql_list_tables - список таблиц во FrontBase-БД.

Описание

resource fbsql_list_tables (string database [, resource link_identifier])

fbsql_list_tables() принимает имя БД и возвращает результирующий указатель, как и функция fbsql_db_query(). Функция fbsql_tablename() должна использоваться для извлечения реальных имён таблиц из результирующего указателя.

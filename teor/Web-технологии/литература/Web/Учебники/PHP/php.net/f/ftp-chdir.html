ftp_chdir

(PHP 3>= 3.0.13, PHP 4)

ftp_chdir - меняет директорию на FTP-сервере.

Описание

bool ftp_chdir (resource ftp_stream, string directory)

Переходит в специфицированную directory.

Возвращает TRUE при успехе, FALSE при ошибке.

asin

(PHP 3, PHP 4)

asin - арксинус.

Описание

float asin (float arg)

Возвращает арксинус аргумента arg  в радианах. asin() это дополнительная функция к sin(), т.е. a==sin(asin(a)) для каждого значения в диапазоне asin().

См. также asinh(), acos() и atan().

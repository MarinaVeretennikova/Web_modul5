Ora_pLogon

(PHP 3, PHP 4)

Ora_pLogon - открывает постоянное соединение Oracle.

Описание

int ora_plogon (string user, string password)

Устанавливает постоянное соединение между PHP и БД Oracle с заданными username и password.

См. также ora_logon().

ob_get_contents

(PHP 4)

ob_get_contents - возвращает содержимое буфера вывода.

Описание

string ob_get_contents (void)

Возвращает содержимое буфера вывода или FALSE, если буферизация вывода неактивна.

См. также ob_start() и ob_get_length().

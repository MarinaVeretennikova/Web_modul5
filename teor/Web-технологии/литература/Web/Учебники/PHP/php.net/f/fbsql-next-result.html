fbsql_next_result

(PHP 4 >= 4.0.6)

fbsql_next_result - перемещает внутренний указатель результата на следующий результат.

Описание

bool fbsql_next_result (resource result_id)

Отправка более одного оператора SQL на сервер или выполнение хранимой процедуры с несколькими результатами заставляет сервер возвращать несколько результирующих наборов. Эта функция тестирует  на наличие дополнительных результатов, имеющихся на сервере. Если дополнительный результирующий набор имеется, она освобождает имеющийся результирующий набор и готовится к извлечению слов из нового результирующего набора. Функция возвращает TRUE, если дополнительный результирующий набор был доступен, и FALSE в противном случае.

Пример 1. fbsql_next_result()
<?php
    $link = fbsql_connect ("localhost", "_SYSTEM", "secret");
    fbsql_select_db("MyDB", $link);
    $SQL = "Select * from table1; select * from table2;";
    $rs = fbsql_query($SQL, $link);
    do {
        while ($row = fbsql_fetch_row($rs)) {
        }
    } while (fbsql_next_result($rs));
    fbsql_free_result($rs);
    fbsql_close ($link);
?>

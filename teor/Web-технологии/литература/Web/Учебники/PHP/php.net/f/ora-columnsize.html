Ora_ColumnSize

(PHP 3, PHP 4)

Ora_ColumnSize - получает размер результирующего столбца Oracle.

Описание

int Ora_ColumnSize (int cursor, int column)

Возвращает размер Oracle-столбца column в курсоре cursor. Столбец 0 это первый столбец.

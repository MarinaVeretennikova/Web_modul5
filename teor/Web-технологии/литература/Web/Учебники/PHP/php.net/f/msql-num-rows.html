msql_num_rows

(PHP 3, PHP 4)

msql_num_rows - получает количество рядов в результате.

Описание

int msql_num_rows (int query_identifier)

msql_num_rows() получает количество рядов в результирующем наборе.

См. также msql(), msql_query() и msql_fetch_row().

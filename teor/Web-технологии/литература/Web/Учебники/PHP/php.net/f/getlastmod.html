getlastmod

(PHP 3, PHP 4)

getlastmod - получает время последней модификации страницы.

Описание

int getlastmod (void)

Возвращает время последней модификации текущей страницы. Возвращается Unix timestamp, подходящий для использования в date(). Возвращает FALSE при ошибке.
Пример 1. getlastmod()
// выведет 'Last modified: March 04 1998 20:43:59.'
echo "Last modified: " . date ("F d Y H:i:s.", getlastmod());

См. также date(), getmyuid(), getmygid(), get_current_user(), getmyinode() и getmypid().

hw_Root

(PHP 3>= 3.0.3, PHP 4)

hw_Root - root object id.

Описание

int hw_root ( )

Возвращает ID объекта hyperroot-коллекции. В настоящее время это всегда 0. Дочерняя коллекция hyperroot является root-коллекцией соединённого сервера.

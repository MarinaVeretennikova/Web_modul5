ldap_first_attribute

(PHP 3, PHP 4)

ldap_first_attribute - возвращает первый атрибут.

Описание

string ldap_first_attribute (resource link_identifier, resource result_entry_identifier, int ber_identifier)

Возвращает первый атрибут во вхождении при успехе, FALSE при ошибке.

По аналогии с вхождениями, атрибуты также читаются один за другим из определённого вхождения. ldap_first_attribute() возвращает первый атрибут во вхождении, на которое указывает result_entry_identifier. Остальные атрибуты запрашиваются последующими вызовами функции ldap_next_attribute().
ber_identifier это идентификатор указателя внутренней памяти. Он передаётся по ссылке. Тот же самый ber_identifier передаётся функции ldap_next_attribute(), которая модифицирует этот указатель.

См. также ldap_get_attributes().

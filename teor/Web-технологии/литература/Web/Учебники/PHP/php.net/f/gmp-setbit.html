gmp_setbit

(PHP 4 >= 4.0.4)

gmp_setbit - устанавливает бит.

Описание

resource gmp_setbit (resource &a, int index [, bool set_clear])

Устанавливает бит index в a.
set_clear определяет, устанавливается в бит 0 или 1. По умолчанию в бит устанавливается 1.

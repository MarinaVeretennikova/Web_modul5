odbc_commit

(PHP 3>= 3.0.6, PHP 4)

odbc_commit - подтверждает ODBC-транзакцию.

Описание

bool odbc_commit (resource connection_id)

Возвращает TRUE при успехе, FALSE при неудаче. Все запущенные транзакции на соединении connection_id подтверждаются.

pdf_open

(PHP 3>= 3.0.6, PHP 4)

pdf_open - не рекомендуется: открывает новый pdf-документ.

Описание

pdf_open() не рекомендуется, используйте pdf_new() плюс pdf_open_file().

См. также pdf_new(), pdf_open_file().

fbsql_list_dbs

(PHP 4 >= 4.0.6)

fbsql_list_dbs - список БД, доступных на FrontBase-сервере.

Описание

resource fbsql_list_dbs ([resource link_identifier])

fbsql_list_dbs() возвращает результирующий указатель, содержащий базы данных, доступные на текущем fbsql-демоне. Используйте fbsql_tablename() для прохода этим указателем результата.

Пример 1. fbsql_list_dbs()
$link = fbsql_connect('localhost', 'myname', 'secret');
$db_list = fbsql_list_dbs($link);

while ($row = fbsql_fetch_object($db_list)) {
    echo $row->Database . "\n";
}

Это пример даст на выводе: 
database1
database2
database3
...

Примечание: вышеприведённый код будет так же хорошо работать с fbsql_fetch_row() или другими подобными функциями.

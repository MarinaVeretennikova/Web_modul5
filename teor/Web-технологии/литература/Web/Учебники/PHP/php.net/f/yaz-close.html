yaz_close

(PHP 4 >= 4.0.1)

yaz_close - закрывает YAZ-соединение.

Описание

int yaz_close (int id)

Закрывает Z-ассоциацию, заданную параметром id. Этот id является целевым ID, возвращённым из предыдущим вызовом команды yaz_connect().

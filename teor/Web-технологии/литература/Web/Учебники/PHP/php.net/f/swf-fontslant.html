swf_fontslant

(PHP 4)

swf_fontslant - устанавливает наклон шрифта.

Описание

void swf_fontslant (float slant)

Устанавливает наклон текущего шрифта на угол slant. Положительные значения - наклон вперёд, отрицательные - назад.

gmp_random

(PHP 4 >= 4.0.4)

gmp_random - случайное число.

Описание

resource gmp_random (int limiter)

Генерирует случайное число. Это число будет в диапазоне от limiter до нуля. Если limiter отрицательный, генерируются отрицательные числа.

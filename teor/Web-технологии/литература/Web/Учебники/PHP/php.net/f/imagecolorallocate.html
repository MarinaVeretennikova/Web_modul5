imagecolorallocate

(PHP 3, PHP 4)

imagecolorallocate - выделение цвета для изображения.

Описание

int imagecolorallocate (resource image, int red, int green, int blue)

imagecolorallocate() возвращает идентификатор цвета, представляющий цвет, составленный из данных RGB-компонентов. Аргумент im возвращается от imagecreate().
red, green и blue это значения красного, зелёного и синего компонентов запрашиваемого цвета. Эти параметры являются целыми числами в диапазоне от 0 до 255. imagecolorallocate() обязана вызываться для создания каждого цвета, используемого в изображении image.
$white = imagecolorallocate ($im, 255, 255, 255);
$black = imagecolorallocate ($im, 0, 0, 0);

Возвращает -1, если выделение не удалось.

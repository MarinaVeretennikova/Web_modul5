get_defined_constants

(PHP 4 >= 4.1.0)

get_defined_constants - возвращает ассоциативный массив из имён всех констант и их значений.

Описание

array get_defined_constants (void)

Эта функция возвращает имена и значения всех определённых в данный момент констант. Сюда входят константы, созданные расширениями, а также константы, созданные функцией define().

Например эта строка: 
print_r (get_defined_constants());

напечатает что-то вроде:

Array
(
    [E_ERROR] => 1
    [E_WARNING] => 2
    [E_PARSE] => 4
    [E_NOTICE] => 8
    [E_CORE_ERROR] => 16
    [E_CORE_WARNING] => 32
    [E_COMPILE_ERROR] => 64
    [E_COMPILE_WARNING] => 128
    [E_USER_ERROR] => 256
    [E_USER_WARNING] => 512
    [E_USER_NOTICE] => 1024
    [E_ALL] => 2047
    [TRUE] => 1
)

См. также get_loaded_extensions(), get_defined_functions() и get_defined_vars().

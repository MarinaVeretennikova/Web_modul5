cpdf_setmiterlimit

(PHP 3>= 3.0.8, PHP 4)

cpdf_setmiterlimit - устанавливает ограничение miter.

Описание

void cpdf_setmiterlimit (int pdf document, float value)

Функция cpdf_setmiterlimit() устанавливает ограничение miter в значение больше или равно 1.

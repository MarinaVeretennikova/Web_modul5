is_string

(PHP 3, PHP 4)

is_string - определяет, является ли переменная строкой.

Описание

bool is_string (mixed var)

Возвращает TRUE, если var является string, иначе FALSE.

См. также is_bool(), is_int(), is_integer(), is_float(), is_real(), is_object() и is_array().

ldap_explode_dn

(PHP 3, PHP 4)

ldap_explode_dn - разделяет DN на части-компоненты.

Описание

array ldap_explode_dn (string dn, int with_attrib)

ldap_explode_dn() используется для разделения DN, возвращённого функцией ldap_get_dn(), на компоненты. Каждая часть известна как Relative Distinguished Name/Относительное Отличительное Имя, или RDN. ldap_explode_dn()возвращает массив из всех этих компонентов.
with_attrib используется для запроса: возвращены RDNs только со значениями, или также и с атрибутами. Для получения RDNs с атрибутами (т.е. в формате attribute=value) установите в with_attrib значение 0, а для получения только значений - установите 1.

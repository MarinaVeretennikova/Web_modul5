link

(PHP 3, PHP 4)

link - создаёт жёсткую ссылку.

Описание

int link (string target, string link)

link() создаёт жёсткую ссылку.

См. также symlink() для создания мягких ссылок и readlink() с linkinfo().

Примечание: эта функция не реализована для платформ Windows.

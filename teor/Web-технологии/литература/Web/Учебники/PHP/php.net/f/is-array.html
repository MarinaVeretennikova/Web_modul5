is_array

(PHP 3, PHP 4)

is_array - определяет, является ли переменная массивом.

Описание

bool is_array (mixed var)

Возвращает TRUE, если var является array, FALSE в ином случае.

См. также is_float(), is_int(), is_integer(), is_string() и is_object().

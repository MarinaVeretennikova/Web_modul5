is_bool

(PHP 4)

is_bool - определяет, является ли переменная булевым значением.

Описание

bool is_bool (mixed var)

Возвращает TRUE, если параметр var является boolean.

См. также is_array(), is_float(), is_int(), is_integer(), is_string() и is_object().

ini_restore

(PHP 4)

ini_restore - восстанавливает значение опции конфигурации.

Описание

string ini_restore (string varname)

Восстанавливает оригинальное значение данной опции конфигурации.

См. также ini_alter(), ini_get(), ini_get_all() и ini_set().

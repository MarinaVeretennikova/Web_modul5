array_push

(PHP 4)

array_push - вставляет один или более элементов в конец массива.

Описание

int array_push (array array, mixed var [, mixed ...])

array_push() рассматривает массив array как стэк и помещает переданные переменные в конец array. Размер array увеличивается на количество добавленных переменных. Имеет тот же эффект, что: 
$array[] = $var;

повторённое для каждой var.

Возвращает новое количество элементов массива.

Пример 1. array_push()
$stack = array ("orange", "banana");
array_push ($stack, "apple", "raspberry");

В результате выполнения этого примера переменная $stack будет содержать следующие элементы: 
Array
(
    [0] => orange
    [1] => banana
    [2] => apple
    [3] => raspberry
)

См. также array_pop(), array_shift() и array_unshift().

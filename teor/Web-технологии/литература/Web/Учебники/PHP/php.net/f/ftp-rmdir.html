ftp_rmdir

(PHP 3>= 3.0.13, PHP 4)

ftp_rmdir - удаляет директорию.

Описание

bool ftp_rmdir (resource ftp_stream, string directory)

Удаляет специфицированную directory.

Возвращает TRUE при успехе, FALSE при ошибке.

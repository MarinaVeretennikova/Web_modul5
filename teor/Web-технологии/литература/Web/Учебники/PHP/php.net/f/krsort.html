krsort

(PHP 3>= 3.0.13, PHP 4)

krsort - сортирует массив по ключам в обратном порядке.

Описание

int krsort (array array [, int sort_flags])

Сортирует массив по ключам в обратном порядке, сохраняя корреляцию ключей с данными. Это используется в основном в ассоциативных массивах.
Пример 1. krsort()
$fruits = array ("d"=>"lemon", "a"=>"orange", "b"=>"banana", "c"=>"apple");
krsort ($fruits);
reset ($fruits);
while (list ($key, $val) = each ($fruits)) {
    echo "$key = $val\n";
}

Этот пример выведет:

d = lemon
c = apple
b = banana
a = orange

Вы можете модифицировать поведение сортировки, используя необязательный параметр sort_flags, детали см. в sort().

См. также asort(), arsort(), ksort(), sort(), natsort() и rsort().

pg_port

(PHP 3, PHP 4)

pg_port - возвращает номер порта, ассоциированный с данным соединением.

Описание

int pg_port (resource connection)

pg_port() возвращает номер порта, по которому соединён данный PostgreSQL connection-ресурс.

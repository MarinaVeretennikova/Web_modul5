ob_get_length

(PHP 4 >= 4.0.2)

ob_get_length - возвращает размер буфера вывода.

Описание

string ob_get_length (void)

Возвращает размер содержимого в буфере вывода или FALSE, если буферизация вывода неактивна.

См. также ob_start() и ob_get_contents().

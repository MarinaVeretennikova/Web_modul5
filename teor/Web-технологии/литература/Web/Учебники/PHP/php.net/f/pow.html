pow

(PHP 3, PHP 4)

pow - экспоненциальное выражение.

Описание

number pow (number base, number exp)

Возвращает base, возведённую в степень exp. Если возможно, эта функция возвращает integer.

Если возведение в степень не может быть вычислено, выдаётся предупреждение, а pow() возвращает FALSE.

Пример 1. Некоторые примеры использования pow()
<?php

var_dump( pow(2,8) ); // int(256)
echo pow(-1,20); // 1
echo pow(0, 0); // 1

echo pow(-1, 5.5); // ошибка

?>

Предупреждение!

В PHP 4.0.6 и ранее pow() всегда возвращает float и не выдаёт предупреждений.

См. также exp() и sqrt().

mysql_get_client_info

(PHP 4 >= 4.0.5)

mysql_get_client_info - получает информацию MySQL-клиента.

Описание

string mysql_get_client_info (void)

mysql_get_client_info() возвращает строку с версией клиентской библиотеки.

Пример 1. mysql_get_client_info
<?php
    printf ("MySQL client info: %s\n", mysql_get_client_info());
?>

На выводе это пример даст:

MySQL client info: 3.23.39

См. также mysql_get_host_info(), mysql_get_proto_info() и mysql_get_server_info().

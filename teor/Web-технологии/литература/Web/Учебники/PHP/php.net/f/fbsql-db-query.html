fbsql_db_query

(PHP 4 >= 4.0.6)

fbsql_db_query - отправляет FrontBase query/запрос.

Описание

resource fbsql_db_query (string database, string query [, resource link_identifier])

Возвращает положительный результирующий идентификатор FrontBase для результата запроса или FALSE при ошибке.

fbsql_db_query() выбирает БД и выполняет запрос к ней. Если необязательный идентификатор ссылки не предоставлен, функция пытается найти открытую ссылку на FrontBase-сервер и, если такой ссылки нет, пытается создать ссылку, как при вызове fbsql_connect() без аргументов.

См. также fbsql_connect().

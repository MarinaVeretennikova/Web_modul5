addslashes

(PHP 3, PHP 4)

addslashes - закавычивает строку слэшами.

Описание

string addslashes (string str)

Возвращает строку со слэшами перед символами, которые должны быть закавычены в запросах БД и т.п.. Это одинарная ('), и двойная (") кавычки, обратный слэш/backslash (\)и NUL (NULL-байт).

Примечание: magic_quotes_gpc включены/ON по умолчанию.

См. также stripslashes(), htmlspecialchars() и quotemeta().

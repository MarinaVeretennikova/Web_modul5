dbase_pack

(PHP 3, PHP 4)

dbase_pack - упаковывает БД dBase.

Описание

bool dbase_pack (int dbase_identifier)

Упаковывает специфицированную БД (безвозвратно удаляя все записи, помеченные для удаления с использованием dbase_delete_record()).

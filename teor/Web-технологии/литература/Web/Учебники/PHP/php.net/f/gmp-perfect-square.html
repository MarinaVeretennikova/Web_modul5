gmp_perfect_square

(PHP 4 >= 4.0.4)

gmp_perfect_square - проверка точности квадрата.

Описание

bool gmp_perfect_square (resource a)

Возвращает TRUE, если a является точным квадратом, FALSE - в ином случае.

См. также: gmp_sqrt(), gmp_sqrtrm().

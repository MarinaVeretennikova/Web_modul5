fbsql_field_name

(PHP 4 >= 4.0.6)

fbsql_field_name - получает имя специфицированного поля в результате.

Описание

string fbsql_field_name (resource result, int field_index)

fbsql_field_name() возвращает специфицированного индекса поля.
result обязан быть правильным идентификатором результата, а field_index это числовое смещение поля.

Примечание: field_index стартует с 0, например, индекс третьего поля это 2, индекс четвёртого - 3 и т.д.

Пример 1. fbsql_field_name() 
// Таблица пользователей состоит из трёх полей:
//   user_id
//   username
//   password

$res = fbsql_db_query("users", "select * from users", $link);

echo fbsql_field_name($res, 0) . "\n";
echo fbsql_field_name($res, 2);

Этот пример даст на выводе:
user_id
password

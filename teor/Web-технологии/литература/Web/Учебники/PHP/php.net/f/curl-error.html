curl_error

(PHP 4 >= 4.0.3)

curl_error - возвращает строку содержащую номер последней ошибки для текущей сессии.

Описание

string curl_error (resource ch)

Предупреждение!

Эта функция в настоящее время ещё не задокументирована; имеется только список аргументов.

fbsql_create_db

(PHP 4 >= 4.0.6)

fbsql_create_db - создаёт БД FrontBase.

Описание

bool fbsql_create_db (string database name [, resource link_identifier])

fbsql_create_db() пытается создать новую БД на сервере, ассоциированном со специфицированным идентификатором ссылки.

Пример 1. fbsql_create_db()
<?php
    $link = fbsql_pconnect ("localhost", "_SYSTEM", "secret")
        or die ("Could not connect");
    if (fbsql_create_db ("my_db")) {
        print("Database created successfully\n");
    } else {
        printf("Error creating database: %s\n", fbsql_error ());
    }
?>

См. также fbsql_drop_db().

getmypid

(PHP 3, PHP 4)

getmypid - получает ID РНР-процесса.

Описание

int getmypid (void)

Возвращает идентификатор ID текущего процесса РНР или FALSE при ошибке.

Предупреждение!

Process IDs не являются уникальными, поэтому являются источником некоторой энтропии. Мы рекомендуем не полагаться на pid'ы в контексте безопасности.

См. также getmygid(), getmyuid(), get_current_user(), getmyinode() и getlastmod().

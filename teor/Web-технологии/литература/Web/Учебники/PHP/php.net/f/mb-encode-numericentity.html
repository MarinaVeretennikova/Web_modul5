mb_encode_numericentity

(PHP 4 >= 4.0.6)

mb_encode_numericentity - кодирует символы в ссылку числовой строки HTML.

Описание

string mb_encode_numericentity (string str, array convmap [, string encoding])

mb_encode_numericentity() конвертирует специфицированные коды символов строки str в ссылку числовой строки HTML. Возвращает конвертированную строку.

array это массив, специфицирующий кодовую область для конвертации.

encoding это кодировка символов.

Пример 1. convmap
$convmap = array (
 int start_code1, int end_code1, int offset1, int mask1,
 int start_code2, int end_code2, int offset2, int mask2,
 ........
 int start_codeN, int end_codeN, int offsetN, int maskN );
// Специфицирует значение Unicode для start_codeN и end_codeN
// Добавляет offsetN к значению и применяет битовую 'AND' с маской maskN, затем
// конвертирует значение в ссылку числовой строки.
Пример 2. mb_encode_numericentity()
/* Конвертирует левую часть ISO-8859-1 в ссылку числовой строки HTML */
$convmap = array(0x80, 0xff, 0, 0xff);
$str = mb_encode_numericentity($str, $convmap, "ISO-8859-1");

/* Конвертирует пользовательский SJIS-win код в блоке 95-104 в
   ссылку числовой строки */
$convmap = array(
       0xe000, 0xe03e, 0x1040, 0xffff,
       0xe03f, 0xe0bb, 0x1041, 0xffff,
       0xe0bc, 0xe0fa, 0x1084, 0xffff,
       0xe0fb, 0xe177, 0x1085, 0xffff,
       0xe178, 0xe1b6, 0x10c8, 0xffff,
       0xe1b7, 0xe233, 0x10c9, 0xffff,
       0xe234, 0xe272, 0x110c, 0xffff,
       0xe273, 0xe2ef, 0x110d, 0xffff,
       0xe2f0, 0xe32e, 0x1150, 0xffff,
       0xe32f, 0xe3ab, 0x1151, 0xffff );
$str = mb_encode_numericentity($str, $convmap, "sjis-win");

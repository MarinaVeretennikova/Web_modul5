sin

(PHP 3, PHP 4)

sin - синус.

Описание

float sin (float arg)

sin() возвращает синус аргумента arg. Параметр arg задан в радианах. 

<?php

// Точность зависит от вашей директивы precision
print sin(deg2rad(60));  //  0.866025403 ...
print sin(60);           // -0.304810621 ...

?>

См. также asin(), cos(), tan() и deg2rad().

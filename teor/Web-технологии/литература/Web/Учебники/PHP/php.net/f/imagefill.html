imagefill

(PHP 3, PHP 4)

imagefill - заливка.

Описание

int imagefill (resource image, int x, int y, int col)

imagefill() выполняет заливку, начиная с координат x, y (верхний левый угол имеет координаты 0, 0), цветом col на изображении image.

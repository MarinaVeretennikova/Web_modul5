tanh

(PHP 4 >= 4.1.0)

tanh - гиперболический тангенс.

Описание

float tanh (float arg)

Возвращает гиперболический тангенс аргумента arg, определённый как sinh(arg)/cosh(arg).

См. также tan(), atanh(), sin() и cos().

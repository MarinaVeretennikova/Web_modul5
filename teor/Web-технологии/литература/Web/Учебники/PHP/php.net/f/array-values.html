array_values

(PHP 4)

array_values - возвращает все значения массива.

Описание

array array_values (array input)

array_values() возвращает все значения массива input.

Пример 1. array_values()
$array = array ("size" => "XL", "color" => "gold");
print_r(array_values ($array));

Это даст на выходе: 
Array
(
    [0] => XL
    [1] => gold
)

Примечание: эта функция была добавлена в PHP 4, а ниже дана реализация для PHP 3.
Пример 2. Реализация array_values() для пользователей PHP 3
function array_values ($arr) {
    $t = array();
    while (list($k, $v) = each ($arr)) {
        $t[] = $v;
    }
    return $t;
}

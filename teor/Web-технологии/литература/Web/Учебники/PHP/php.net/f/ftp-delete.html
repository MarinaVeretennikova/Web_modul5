ftp_delete

(PHP 3>= 3.0.13, PHP 4)

ftp_delete - удаляет файл на FTP-сервере.

Описание

bool ftp_delete (resource ftp_stream, string path)

ftp_delete() удаляет файл, специфицированный в path, с FTP-сервера.

Возвращает TRUE при успехе, FALSE при ошибке.

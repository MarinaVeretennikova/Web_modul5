pg_lo_tell

(PHP 4 >= 4.2.0)

pg_lo_tell - возвращает текущую позицию большого объекта/large object.

Описание

int pg_lo_tell (resource large_object)

pg_lo_tell() возвращает текущую позицию (смещение от начала большого объекта).

См. также pg_lo_seek().

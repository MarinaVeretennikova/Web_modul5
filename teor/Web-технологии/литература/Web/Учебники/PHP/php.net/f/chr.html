chr

(PHP 3, PHP 4)

chr - возвращает специфический символ.

Описание

string chr (int ascii)

Возвращает строку из одного символа, содержащую символ, специфицированный параметром ascii.
Пример 1. chr()
$str .= chr(27); /* символ escape в конце $str */

/* Чаще используют: */

$str = sprintf("The string ends in escape: %c", 27);

Таблицу ASCII-символов вы можете найти здесь: http://www.mindspring.com/~jc1/serial/Resources/ASCII.html.

Эта функция дополняет ord(). См. также sprintf() со строкой формата %c.

ircg_is_conn_alive

(PHP 4 >= 4.0.5)

ircg_is_conn_alive - проверяет статус соединения.

Описание

boolean ircg_is_conn_alive (resource connection)

ircg_is_conn_alive() возвращает TRUE, если соединение connection работает, и FALSE, если сервер не отвечает.

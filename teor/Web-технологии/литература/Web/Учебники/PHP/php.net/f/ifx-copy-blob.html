ifx_copy_blob

(PHP 3>= 3.0.4, PHP 4)

ifx_copy_blob - дублирует данный blob-объект.

Описание

int ifx_copy_blob (int bid)

Дублирует данный blob-объект. bid это ID этого blob-объекта.

Возвращает FALSE при ошибке, иначе id нового blob-объекта.

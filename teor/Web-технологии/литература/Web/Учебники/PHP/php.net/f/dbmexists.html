dbmexists

(PHP 3, PHP 4)

dbmexists - сообщает, существует ли значение для ключа в DBM-БД.

Описание

bool dbmexists (resource dbm_identifier, string key)

Возвращает TRUE, если имеется значение ассоциированное с ключом key.

fbsql_insert_id

(PHP 4 >= 4.0.6)

fbsql_insert_id - получает id, сгенерированный предыдущей операцией INSERT.

Описание

int fbsql_insert_id ([resource link_identifier])

fbsql_insert_id() возвращает ID, сгенерированный для столбца, определённого как DEFAULT UNIQUE предыдущим запросом INSERT, с использованием данного link_identifier. Если link_identifier не специфицирован, принимается последняя открытая ссылка.

fbsql_insert_id() возвращает 0, если предыдущий запрос не генерирует значение DEFAULT UNIQUE. Если вам нужно сохранить значение на будущее, вызовите fbsql_insert_id() сразу после запроса, генерирующего это значение.

Примечание: значение FrontBase SQL-функции LAST_INSERT_ID() всегда содержит самое последнее сгенерированное DEFAULT UNIQUE и не восстанавливается между запросами.

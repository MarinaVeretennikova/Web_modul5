OCIFetch

(PHP 3>= 3.0.4, PHP 4)

OCIFetch - извлекает следующий ряд в буфер результата.

Описание

int OCIFetch (int statement)

OCIFetch() извлекает следующий ряд (для операторов SELECT) во внутренний буфер результата.

fdf_create

(PHP 3>= 3.0.6, PHP 4)

fdf_create - создаёт новый FDF-документ.

Описание

int fdf_create (void)

fdf_create() создаёт новый FDF-документ. Эта функция нужна, если необходимо заполнить поля ввода FDF-документа данными.

Пример 1. Заполнение FDF-документа
<?php
$outfdf = fdf_create();
fdf_set_value($outfdf, "volume", $volume, 0);

fdf_set_file($outfdf, "http:/testfdf/resultlabel.pdf");
fdf_save($outfdf, "outtest.fdf");
fdf_close($outfdf);
Header("Content-type: application/vnd.fdf");
$fp = fopen("outtest.fdf", "r");
fpassthru($fp);
unlink("outtest.fdf");
?>

См. также fdf_close(), fdf_save(), fdf_open().

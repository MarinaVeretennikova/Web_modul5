fbsql_list_fields

(PHP 4 >= 4.0.6)

fbsql_list_fields - список результирующих полей FrontBase.

Описание

resource fbsql_list_fields (string database_name, string table_name [, resource link_identifier])

fbsql_list_fields() возвращает информацию о данной таблице tablename. Аргументами являются имя БД и имя таблицы. Возвращается результирующий указатель, который можно использовать с функциями fbsql_field_flags(), fbsql_field_len(), fbsql_field_name() и fbsql_field_type().

Результирующий идентификатор это положительное целое число. Функция возвращает -1, если возникла ошибка. Строка с описанием ошибки помещается в $phperrmsg, и, если функция не была вызвана как @fbsql(), то эта строка ошибки также будет напечатана.

Пример 1. fbsql_list_fields()
$link = fbsql_connect('localhost', 'myname', 'secret');

$fields = fbsql_list_fields("database1", "table1", $link);
$columns = fbsql_num_fields($fields);

for ($i = 0; $i < $columns; $i++) {
    echo fbsql_field_name($fields, $i) . "\n";;
}

Пример даст на выходе: 
field1
field2
field3
...

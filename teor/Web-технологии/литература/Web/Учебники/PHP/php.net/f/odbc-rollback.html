odbc_rollback

(PHP 3>= 3.0.6, PHP 4)

odbc_rollback - откатывает транзакцию.

Описание

int odbc_rollback (resource connection_id)

Откатывает все запущенные операторы в соединении connection_id. Возвращает TRUE при успехе, FALSE при неудаче.

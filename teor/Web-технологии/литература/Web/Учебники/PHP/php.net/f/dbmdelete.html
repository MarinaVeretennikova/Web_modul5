dbmdelete

(PHP 3, PHP 4)

dbmdelete - удаляет значение для данного ключа/key из БД DBM.

Описание

bool dbmdelete (resource dbm_identifier, string key)

Удаляет значение данного key в БД.

Возвращает FALSE, если этот key не существует в БД.

gmmktime

(PHP 3, PHP 4)

gmmktime - получает UNIX timestamp для GMT-даты.

Описание

int gmmktime (int hour, int minute, int second, int month, int day, int year [, int is_dst])

Идентична mktime(), но передаваемые параметры представляют GMT-дату.

ibase_commit

(PHP 3>= 3.0.7, PHP 4)

ibase_commit - подтверждает транзакцию.

Описание

int ibase_commit ([int link_identifier, int trans_number])

Подтверждает транзакцию trans_number, которая была создана функцией ibase_trans().

shm_remove

(PHP 3>= 3.0.6, PHP 4)

shm_remove - удаляет shared-память с систем Unix.

Описание

int shm_remove (int shm_identifier)

Удаляет shared-память с систем Unix. Все данные разрушаются.

Примечание: эта функция не работает в системах Windows.

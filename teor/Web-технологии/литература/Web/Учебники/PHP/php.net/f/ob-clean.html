ob_clean

(PHP 4 >= 4.2.0)

ob_clean - очищает буфер вывода.

Описание

void ob_clean (void)

Эта функция удаляет содержимое буфера вывода.

Эта функция не разрушает буфер вывода, как это делает ob_end_clean().

См. также ob_flush(), ob_end_flush() и ob_end_clean().

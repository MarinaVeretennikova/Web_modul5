dechex

(PHP 3, PHP 4)

dechex -- 10-ричное в 16-ричное.

Описание

string dechex (int number)

Возвращает строку, содержащую 16-ричное представление аргумента number. Наибольшее допустимое число для конвертирования - 10-ричное 2147483647, дающее "7fffffff" в 16-ричном виде.

См. также hexdec().

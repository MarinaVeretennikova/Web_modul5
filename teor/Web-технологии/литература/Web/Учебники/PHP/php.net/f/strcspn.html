strcspn

(PHP 3>= 3.0.3, PHP 4)

strcspn - находит длину начального сегмента, не совпадающего с маской.

Описание

int strcspn (string str1, string str2)

Возвращает длину начального сегмента строки str1, который не содержит никаких символов из str2.

См. также strspn().

ftp_systype

(PHP 3>= 3.0.13, PHP 4)

ftp_systype - возвращает идентификатор системного типа удалённого FTPсервера.

Описание

string ftp_systype (resource ftp_stream)

Возвращает тип удалённой системы или FALSE при ошибке.

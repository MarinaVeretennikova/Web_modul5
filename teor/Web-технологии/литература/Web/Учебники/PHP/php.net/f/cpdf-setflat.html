cpdf_setflat

(PHP 3>= 3.0.8, PHP 4)

cpdf_setflat - устанавливает плоскость/flatness.

Описание

void cpdf_setflat (int pdf document, float value)

Функция cpdf_setflat() устанавливает плоскость ("ровность")/flatness в значение от 0 до 100.

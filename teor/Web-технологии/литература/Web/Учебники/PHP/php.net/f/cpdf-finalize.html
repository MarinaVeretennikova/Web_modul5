cpdf_finalize

(PHP 3>= 3.0.8, PHP 4)

cpdf_finalize - заканчивает документ.

Описание

void cpdf_finalize (int pdf document)

Функция cpdf_finalize() заканчивает документ. Вы должны по прежнему вызывать cpdf_close().

См. также cpdf_close().

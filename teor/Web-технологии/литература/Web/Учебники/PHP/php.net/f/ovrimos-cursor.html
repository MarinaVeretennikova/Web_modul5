ovrimos_cursor

(PHP 4 >= 4.0.3)

ovrimos_cursor - возвращает имя курсора.

Описание

int ovrimos_cursor (int result_id)

ovrimos_cursor() возвращает имя курсора. Используется, когда необходимо выполнить позиционированные update или delete.

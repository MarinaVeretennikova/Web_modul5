SUP - надстрочный знак (Нет в HTML 2.0!)

Цель

Для представления надстрочного знака, который может быть показателем в выражении или индексом переменных.

Типичное отображение

Немного выше уровня нормального текста и, возможно, малым шрифтом. Смотрите общие примечания к отображению разметки.

Основной синтаксис

<SUP>текст</SUP>

Возможные атрибуты

Никаких.

Допустимый контекст

Текстовый контейнер, т.е. любой элемент, который может содержать текстовые элементы. Это включает большинство элементов HTML. Текстовые элементы могут быть вложенными.

Содержимое

Текстовые элементы. Перевод абзацев не разрешен.

Примеры

Пример SUP-1.html: 


The notation A<SUP>T</SUP> denotes the transpose of A.

Пример SUP-2.html: 


Consider the equation

x<SUP>n</SUP> + y<SUP>n</SUP> = z<SUP>n</SUP>.

Пример SUP-3.html: 


The expression a<SUP>b<SUP>c</SUP></SUP>

means a<SUP>(b<SUP>c</SUP>)</SUP>.

Пример SUP-4.html: 


This example is a text paragraph which contains several

superscripted expressions such as m<SUP>2</SUP> and e<SUP>x</SUP>.

They may affect the visual appearance of the paragraph by

forcing the browser to use different line heighths. This

applies in particular to expressions with large and nested

superscripts such as (f(a))<SUP>e<SUP>x<SUP>2y</SUP></SUP></SUP>.

Примечания

Цифры 1, 2 или 3 в качестве надстрочных символов можно представить и другим способом, так как набор символов ISO Latin 1 содержит коды для них. Например: mІ или, с использованием escape символов, m&sup2;.

Существует также тег для подстрочного знака SUB, однако HTML 3.2 не обеспечивает полной поддержки математических формул.

Так как этот тег - новый, поддерживают его не все программы. Некоторые броузеры просто игнорируют его, представляя, например, a<SUP>T</SUP> как aT. Текстовые броузеры также не могут правильно использовать SUP.

Как видно из последнего примера, надстрочные символы могут быть вложенными, что приводит к отображению внутреннего надстрочного текста очень маленьким шрифтом. Броузер Internet Explorer игнорирует теги SUP для уровня вложенности выше двух.

PARAM - параметры апплета (Нет в HTML 2.0!)

Цель

Передать параметры Java апплету.

Типичное отображение

Непосредственно не отображается, но может проявиться в действиях апплета.

Основной синтаксис

<PARAM NAME=имя VALUE=значение>

Возможные атрибуты

имя атрибутавозможные значениясмыслпримечания
NAMEимя имя параметраобязательно
VALUEстроказначение параметра 

Допустимый контекст

Элемент APPLET

Содержимое

Никакое.

Примеры


<PARAM NAME=snd VALUE="Hello.au">

Примечания

Символы escape последовательностей, например, &eacute;, &quot; и &#185; определяются прежде, чем значение параметра передается апплету. Для включения символа & используйте &amp;.

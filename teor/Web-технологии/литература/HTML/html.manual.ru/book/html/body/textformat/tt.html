TT

(HTML 2.0) – Tele Type

Данный элемент отображает помещенный между начальным и конечным тегами текст моноширинным шрифтом.


  Пример:
      <TT>СЕНЯ ЗПТ ТВОЮ МАТЬ ТЧК ПОДРОБНОСТИ В ПИСЬМЕ ТЧК</TT>

Верстка HTML-документов при разработке веб-сайтов Оглавление Общие требования к веб-сайту.

Определения.

В данной главе рассматриваются проблемы и особенности верстки отдельных HTML-документов и веб-сайта в целом. Начнем с некоторых определений, чтобы работать в одной системе координат.

/index.html
Главная страница сайта.
index.html
Главный файл в любой директории, который открывается по умолчанию при обращении к данной директории.
/cgi-bin/
Директория с исполняемыми файлами, должна быть закрыта на чтение.
/styles.css
Таблица глобальных каскадных стилей.
/images/
Директория для хранения глобальных картинок веб-сайта.
/images/0.gif
Прозрачный файл размером 1х1 пиксель.
/include/
Директория для хранения общих кусков HTML-кода.
/include/head.inc
Верхний колонтитул.
/include/footer.inc
Нижний колонтитул.
Файл с функциями JavaScript здесь не описан, т.к. JavaScript применяется не очень часто, и набора общих функций для всего сайта может не существовать.

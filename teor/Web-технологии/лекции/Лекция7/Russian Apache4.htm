Где взять?
С октября 1998 года поддержка старых версий, основанных на Apache 1.0.x-1.2.x, прекращена, хотя эти версии все еще доступны. Поддерживается только версия на основе текущей стабильной версии Apache.

Сам сервер Apache можно взять на www.apache.org или на его зеркалах. Необходимые компоненты, реализующие функциональность Russian Apache можно взять по ftp: на ftp-сервере apache.lexa.ru. На этот сервер будут помещаться все последующие версии правок (исправление ошибок, правки для новых версий Apache, новые возможности). В настоящее время доступны:

Apache-1.3.34 RUS Patchlevel 30.22 - версия на основе Apache-1.3.34


  • Apache-1.3.34 + RUS-pl30 patches - полный набор исходных текстов
  • Только патчи, накладываемые поверх Apache-1.3.34 clean source tree

Win-32 версии доступны по адресу ftp://ftp.lexa.ru/pub/apache-rus/win32
Поддержка Win32-версии осуществляется Ростиславом Уральским

Версии на основе Apache-1.3.x, 1.2.6 и 1.1.x - См. архив старых версий.

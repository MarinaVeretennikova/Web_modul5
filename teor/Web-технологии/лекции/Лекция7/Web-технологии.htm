Рабочая программа по специализации "Web-технологии"


Общие положения: Несомненной основой роста компьютерной индустрии является Internet, которая из технического явления переходит в разряд социальных. Предлагаемый курс обучает основным технологическим составляющим Internet’а, определяющим его информационное наполнение.
Курс разделен на две части: основы и профессиональный уровень. В первой части слушателя знакомят с базовыми понятиями Web-программирования (HTML) и объясняют построение Web-приложений, используя клиентские (JavaScript, VBScript, DHTML) технологии. На практических занятиях слушатель испытает себя в роли Web-дизайнера, изучая HTML и CSS, и Web-программиста (JavaScript, VBScript).
Профессиональный уровень обучения предполагает углубленное изучение клиентских (DHTML), смешанных (ActiveX), а также серверных (CGI-программирование, использование Perl, PHP, SSI, ASP) технологий. Основной упор при этом делается на приобретение навыков работы с базами данных и бизнес-проектами. Двухуровневая система подготовки, позволяет “продвинутым” слушателям непосредственно приступить к изучению серверных технологий и нетривиальных вопросов практического использования Perl, и ActiveX и др.
В процессе обучения слушатель:
- получит навыки работы с разнообразными Internet-технологиями, используя новейшие инструментальные комплексы, такие как Visual InterDev 6.0;
- получит опыт работы с последними версиями спецификаций ADO 2.5, Visual Basic Scripting Edition 6.0, Jscript 6.0, CSS 2.0 и др.;
- овладеет навыками практической работы в команде и научится применять полученные знания при выполнении реальных проектов.

Цель подготовки: получить практические навыки разработки Internet-приложений различного назначения.
Задачи подготовки: освоить современные методы и средства разработки Internet-приложений; овладеть приемами разработки приложений в Visual InterDev 6.0; овладеть HTML, CSS 2.0, VBScript, JavaScript, ASP, DHTML, HTTP-протоколом, CGI, SSI, Perl, ActiveX, PHP.
Пре-реквизиты(дисциплины, усвоение которых необходимо для обучения по данной специализации): основы программирования.

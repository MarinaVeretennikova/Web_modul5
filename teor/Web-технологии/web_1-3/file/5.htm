• Создать web-документ, который должен представлять собой домашнюю страничку, содержать информацию о себе, свои интересы и т.д.

  • Страничка должна иметь достаточную смысловую нагрузку, включать в себя вставки рисунков, гипертекстовые ссылки. Должны имется не менее пяти взаимосвязанных страниц, содержащий объекты различного рода.

  • Связь между страницами должна осуществляться путём разнообразных ссылок, включая в себя и графические.

Основные требования:

  1. В качестве основной страницы использовать страницу с фреймовой структурой.
  2. Структуру страниц представить в виде таблиц.
